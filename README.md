# ProjectHS
<b>프로젝트 목적</b> - 이 프로젝트는 단순한 기능 구현을 넘어서, 확장성과 유지보수성을 고려한 구조적 설계에 중점을 두고 개발되었습니다.
명확한 역할 분리를 기반으로 한 아키텍처를 구성함으로써, 협업에 유리하고 재사용이 쉬운 코드 구조를 지향했습니다.
